> Eslint your code with Danger

## Usage

Install:

```sh
yarn add danger-plugin-titanium-module --dev
# OR
npm install danger-plugin-titanium-module -D
```

At a glance:

```js
// dangerfile.js
import lint from 'danger-plugin-titanium-module'

await lint()
```
